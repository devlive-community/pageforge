---
title: 超链接
---

PageForge 支持 Markdown 的默认超链接语法。您可以使用 `[]()` 格式来创建链接，例如 `[链接文本](链接URL)`。

PageForge 实现了更多的语义，支持更多的超链接格式。

## 基本语法

---

```
[链接文本](链接URL)
```

[PageForge](https://pageforge.devlive.org)

## 带标题

---

```
[链接文本](链接URL "可选标题")
```

[PageForge](https://pageforge.devlive.org "PageForge")

## 带打开方式

---

```
[链接文本](链接URL "可选标题" "打开方式")
```
